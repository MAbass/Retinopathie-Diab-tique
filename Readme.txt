Dans notre logiciel on a récupéré notre dataset sur le site du kaggle. Apres la réception des données, on a fait le pré-traitement des données pour pouvoir y travailler puisqu'il avait des données unitiles. Sur le projet on effectué quelques visualisations en visualisant l'image brute elle-même. Ensuite on a fait l'égalisation des pixels pour visualiser le résultat. Pour terminer on a créer quelque modèle pour faire quelque prédiction sur nos données.
